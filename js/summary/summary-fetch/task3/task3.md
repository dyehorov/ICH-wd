# Цель: Создать интерфейс, с двумя панелями: списком альбомов и сеткой фотографий.

Описание:
Приложение должно отображать альбомы, принадлежащие пользователям, и фотографии внутри этих альбомов. Реализовать поиск, модальное окно и "удаление" фотографий.

Сделать запросы к:
/albums
/users
/photos

Объединить альбомы с пользователями, чтобы каждый альбом знал своего владельца (имя пользователя).
Интерфейс с двумя колонками:
Левая колонка: Список альбомов. Для каждого альбома отображать его title и имя владельца.
Правая колонка: Сетка фотографий (thumbnailUrl)

Взаимодействие:
При клике на альбом в левой колонке, правая колонка должна обновляться и показывать только фотографии из этого выбранного альбома.

Над правой колонкой должен отображаться заголовок: "Фотографии альбома: [Название альбома]".

Поиск по фотографиям:
Над сеткой фотографий разместить поле поиска.
Поиск должен работать в рамках выбранного альбома по названию фотографии (title).

Модальное окно:

При клике на миниатюру фотографии открывать модальное (всплывающее) окно.
В модальном окне показывать большую версию изображения (url), его полное название (title) и кнопку "Закрыть".
Реализовать закрытие модального окна по кнопке и по клику на затемненную область вокруг него.

"Удаление" фотографий:
В правой колонке, при наведении на фотографию, должна появляться иконка "корзины".
При клике на корзину фотография должна "удаляться":
(Опционально) Отправить DELETE-запрос на /photos/{id}.
Удалить фотографию из локального массива и из DOM.

API Endpoints:

https://jsonplaceholder.typicode.com/albums
https://jsonplaceholder.typicode.com/users
https://jsonplaceholder.typicode.com/photos
