Вам нужно создать небольшое приложение, которое показывает карточку профиля пользователя.
Cпарва от карточки находится панель с настройками. Изменяя настройки (через обычные поля ввода), карточка профиля должна мгновенно меняться.

Список обязательных компонентов MUI для использования:
Card (Card, CardContent, CardMedia, CardActions)
Avatar
Typography
Button
TextField (или Slider)
Stack (или Box)
Switch (опционально)
Alert

TASK
Создайте React-компонент (например, ProfilePlayground), который состоит из двух колонок (или секций), используя Stack или Grid.

1.  Левая часть: Карточка профиля (Preview)
    Создайте карточку, которая отображает:

Аватар пользователя (используйте Avatar с вложенной картинкой или первую букву имени).
Имя и Фамилия (используйте Typography с разными вариантами variant).
Статус (On/Off), который меняет цвет текста.
Кнопки действий ("Написать сообщение", "Предложить работу").
Alert с каким-нибудь текстом (например, "Идет поиск стажеров"), который появляется только если включен чекбокс "Показать подсказки".

2. Правая часть: Панель управления (Controls)
   Создайте панель, где с помощью разных полей ввода можно менять параметры карточки. Здесь и происходит обучение атрибутам и значениям.

Ваша задача — связать состояние (state) правой части с пропсами левой.
Элемент управления (props для изучения) Назначение Пример использования
TextField (variant, label) Поле ввода имени и фамилии Изменяет текст внутри карточки.
Slider (value, min, max) Регулировка размера аватара Изменяет ширину/высоту аватара через sx проп.
Select (value, onChange) Выбор цвета кнопки Меняет color у кнопки ("primary", "secondary", "success").
Switch (checked, onChange) Вкл/Выкл статус "Онлайн" Меняет цвет статуса на зеленый и добавляет точку перед именем (используйте условный рендеринг).
Radio Group Выбор стиля карточки Меняет variant карточки (elevation / outlined).
Checkbox Показать/Скрыть Alert Управляет отображением компонента Alert.

//////////
Для состояния (state): Создайте один объект в useState, например:
Пример: [profileSettings, setProfileSettings] = useState({ name: 'Иван', surname: 'Иванов', avatarSize: 40, buttonColor: 'primary', isOnline: false, cardVariant: 'elevation', showAlert: false })

Для стилизации: используйте inline стили

Для передачи значений: В компонентах Select, TextField обязательно используйте атрибуты value и onChange.
Пример: <TextField label="Имя" value={profileSettings.name} onChange={(e) => setProfileSettings({...profileSettings, name: e.target.value})} />

........
Что будет меняться визуально при настройке:
Текст в полях → сразу меняется имя в карточке
Слайдер → аватар увеличивается/уменьшается
Radio кнопки → кнопки в карточке меняют цвет (синий, фиолетовый, зеленый)
Switch "Онлайн" →
Выкл: серый текст "○ Офлайн"
Вкл: зеленый текст "● Онлайн" с точкой
Switch "Показать Alert" → синее уведомление внизу карточки появляется/исчезает \*
Аватар плавно меняет размер (анимация)
При наведении карточка приподнимается
Цветовая схема применяется ко всем элементам (кнопки, свитчи, аватар)
Онлайн-статус меняет цвет и иконку
Уведомление пульсирует
Все элементы имеют отступы и тени для красивого вида

Доп. задания:

1 Добавь возможность загружать свою фотографию в карточку профиля.
Добавь кнопку "Загрузить фото" под аватаром
При нажатии на кнопку должно открываться окно выбора файла (только изображения)
После выбора изображения — аватар должен поменяться на выбранную картинку

2 Добавь зеленую точку на аватар, которая показывает онлайн-статус пользователя.
Когда Switch "Онлайн статус" включен — на аватаре должна появиться зеленая точка внизу справа
Когда статус выключен — точки нет
Подсказка: используй компонент Badge с атрибутами:
overlap="circular"
anchorOrigin={{ vertical: "bottom", horizontal: "right" }}

3 Создай Alert, который появляется только при включенном Switch "Показать Alert".
Alert должен содержать текст: "Не забудь загрузить аватарку!"
Добавь в Alert кнопку "Понятно"
При нажатии на кнопку должно появляться всплывающее окно с сообщением "Спасибо, что прочитал!"

4 Добавь под основным Alert'ом блок из четырех маленьких Alert'ов с разными типами.
Создай 4 Alert'а с severity: success, info, warning, error
У каждого должен быть свой текст:
success: "Отлично! MUI работает"
info: "Попробуй изменить цвет кнопок"
warning: "Не забывай про атрибуты"
error: "Ошибок нет, все отлично!"

Комопненты и атрибуты для доп заданий:
Badge - overlap, anchorOrigin, badgeContent
Alert - severity, variant, action, icon
Button - component, startIcon, size
input - type, hidden, accept, onChange
FileReader - onload, readAsDataURL

5. При нажатии на кнопку "Написать" должно появляться уведомление (Alert) с предложением написать сообщение пользователю.
   Добавь обработчик нажатия на кнопку "Написать"
   При клике должно появляться всплывающее окно (alert) с текстом: "Напиши сообщение для {Имя пользователя}"
   Вместо {Имя пользователя} должно подставляться текущее имя и фамилия
   Пример: Если пользователь Анна Петрова, то alert: "Напиши сообщение для Анна Петрова"

6. При нажатии на кнопку "Предложить работу" должно появляться модальное окно с формой.
   Добавь обработчик нажатия на кнопку "Предложить работу"
   При клике должно появляться окно подтверждения (confirm) с вопросом: "Хочешь предложить работу {Имя пользователя}?"
   Если пользователь нажимает "OK" — показывать alert: "Заявка отправлена! {Имя пользователя} получит предложение"
   Если пользователь нажимает "Отмена" — показывать alert: "Отправка отменена"

7. Сделай так, чтобы текст на кнопках зависел от выбранной роли пользователя.
   Если выбрана роль "Разработчик" — кнопка "Предложить работу" меняет текст на "Предложить проект"
   Если выбрана роль "Дизайнер" — кнопка "Предложить работу" меняет текст на "Предложить заказ"
   Если выбрана роль "Менеджер" — кнопка "Предложить работу" меняет текст на "Предложить вакансию"
   Если выбрана роль "Аналитик" — кнопка "Предложить работу" меняет текст на "Предложить стажировку"

8. Уведомление при наведении
   Добавь всплывающую подсказку (tooltip) при наведении на кнопки.

При наведении на кнопку "Написать" — показывать подсказку: "Нажми, чтобы написать сообщение"
При наведении на кнопку "Предложить работу" — показывать подсказку: "Нажми, чтобы отправить предложение"
(используй компонент Tooltip из MUI)
