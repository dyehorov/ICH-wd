Задание 8
Создайте HTTP сервер, который регистрирует несколько обработчиков на одно событие и использует метод once() для одноразового события.
Создайте проект multiHandlerServer и файл server.js
Импортируйте модули http и events
Создайте экземпляр EventEmitter
Зарегистрируйте три разных обработчика на событие userVisit:
Первый обработчик: выводит в консоль "[ЛОГ] Посещение пользователя зафиксировано"
Второй обработчик: выводит в консоль "[СТАТИСТИКА] Увеличиваем счетчик посещений"
Третий обработчик: выводит в консоль "[УВЕДОМЛЕНИЕ] Пользователь зашел на сайт"
Зарегистрируйте одноразовый обработчик (используя once()) на событие firstStart:
Выводит в консоль "[ВАЖНО] Сервер запущен впервые! (это сообщение появится только один раз)"
Создайте HTTP сервер со следующими маршрутами:
Маршрут "/" (корень) – сгенерировать событие userVisit, отправить HTML страницу с ссылками на все доступные маршруты
Маршрут "/about" – сгенерировать событие userVisit, отправить текст "О нас: Мы изучаем EventEmitter в Node.js"
Маршрут "/reload" – сгенерировать событие userVisit, а также сгенерировать событие firstStart, отправить текст "Попытка повторной генерации одноразового события"
Маршрут "/remove" – удалить второй обработчик (со статистикой) с помощью removeListener() и отправить текст "Второй обработчик (статистика) удален"
Сервер должен запускаться на порту 3000 и при запуске сгенерировать событие firstStart (оно должно сработать только один раз за всё время работы сервера)
Для любого другого маршрута вернуть текст "404 Not Found"
