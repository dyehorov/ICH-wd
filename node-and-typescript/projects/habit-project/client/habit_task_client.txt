Общие требования
Использовать React + Vite
Для запросов к API использовать fetch или axios
Все данные получать с бэкенда (никаких локальных массивов)
API base URL: http://localhost:3333

Страницы / Компоненты
1. Главная страница (дашборд)

Показывает общую статистику:
Сколько всего привычек создано
Сколько выполнений отмечено за всё время
Привычка с самой длинной серией (из запроса /stats/longest-streak)
Среднее настроение за всё время (из запроса /stats/mood-correlation)

2. Список привычек

Показывает все привычки из GET /habits
У каждой привычки видно: название, категорию, сложность, текущую серию (streak)
Кнопка "Выполнить" рядом с каждой привычкой — отправляет POST на /habits/:id/complete с полями mood (выпадающий список от 1 до 5) и notes (текстовое поле, необязательное)
Кнопки "Редактировать" и "Удалить"

3. Форма создания привычки
Поля:
Название (текст, обязательное)
Категория (выпадающий список: health, education, productivity, mindfulness)
Сложность (выпадающий список: easy, medium, hard)
Отправляет POST на /habits

4. Страница аналитики (отдельная вкладка или страница)

Отображает результаты всех 8 аналитических запросов в виде карточек или графиков (можно простыми цифрами и текстом):
Серия побед — название привычки и количество дней
Лучший день недели — какой день и сколько выполнений
Месяц-рекордсмен — месяц, год и количество выполнений
Заброшенные привычки — список привычек, которые не трогали больше 7 дней
Корреляция настроения — таблица: сложность, среднее настроение, количество выполнений
Идеальный день — дата, количество привычек, среднее настроение
Золотая середина — привычка, её выполнений и среднее по всем
Выгорание — список привычек у которых streak > totalCompletions

5. Компонент для отображения одной привычки с прогрессом

Показывает название, категорию, сложность, текущую серию (streak), рекордную серию (bestStreak), общее количество выполнений (totalCompletions)
Кнопка "Посмотреть историю" — открывает модальное окно или отдельную страницу со списком всех выполнений (дата, настроение, заметка)


Взаимодействие
При выполнении привычки после успешного POST запроса:
Обновить список привычек (перезагрузить данные с GET /habits)
Обновить дашборд (перезагрузить статистику)
Показать уведомление "Привычка выполнена!"

При удалении привычки:
Подтверждение действия (alert или модальное окно)
После удаления обновить список

При создании привычки:
После успешного создания перенаправить на страницу со списком привычек или закрыть форму и обновить список
